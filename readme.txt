== Description ==

This widget adds the latest features from BuzzFeed.com to your sidebar. 

 
== Installation ==

1. Extract the folder buzzfeed from the zip archive and upload it to wp-content/plugins/
2. Activate the widget through the Plugins menu in WordPress. 
3. Switch to the Options > BuzzFeed tab to tweak the widget as you like. 
   Click the grid to size the widget, choose image size, pick a color, select your favourite categories, etc. 
   You will find an option that lets you add the widget to your sidebar automatically or paste code into the theme editor.  
   
== Features == 

1. Can be added to the sidebar via drag&drop as a widget
2. Graphic configuration tool
3. Advanced template editing makes the widget fully customizable
